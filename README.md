# Ballad Larp
This is the website for Ballad, a larp running Spring-Fall 2017.
- - -
## Run Locally
```shell
npm install
bower install
npm run less-compile
```
Then run a webserver from this directory. For example, `python -m SimpleHTTPServer`.
